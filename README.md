# [BLOCK]: Ежедневный протокол симптомов. Мониторинг психоэмоционального состояния
## что реально имеет смысл фиксировать:

 ### 1. Аффективное состояние (настроение за день):
  * Депрессивное
  * Нейтральное
  * Приподнятое/эйфорическое
  * Раздражительное
 
 ### 2. Энергетический уровень:
  * 0 = полное истощение
  * 5 = более-менее
  * 10 = бодр и активен
 
 ### 3. Сон:
  * Кол-во часов
  * Качество (плохо / нормально / хорошо)
  * Засыпание долго / средне / быстро
  * Просыпался ночью или нет
 
 ### 4. Аппетит:
  * Снижен
  * Нормальный
  * Повышенный
 
 ### 5. Концентрация:
  * 0 = полный туман
  * 5 = кое-как
  * 10 = ясная голова
 
 ### 6. Суицидальные мысли:
  * Нет
  * Есть, но без планов
  * Есть с планами
  * Очень острые / навязчивые
 
 ### 7. Социальная активность:
  * Был ли контакт с кем-то (онлайн / оффлайн)
  * Изоляция полный день
 
 ### 8. Физическая активность:
  * 0 = весь день лежал
  * 5 = ходил по дому
  * 10 = гулял / спорт / что-то активное
 
 ### 9. Ангедония (не способность получать удовольствие):
  * Полная
  * Частичная
  * Нет

 ### 10. Тревожность:
   * Нет
   * Слабая
   * Средняя
   * Сильная

 ### 11. Форма тревоги (опционально):
   * Фоновая (постоянный общий тревожный фон весь день)
   * Панические эпизоды (внезапные пики, как паническая атака)
   * Тревожные мысли / накрутка (постоянные мысленные сценарии)
   * Телесная тревога (дрожь, потливость, сердцебиение, спазмы)
   * Нет
 
 ### 12. Раздражительность / Агрессия:
   * Нет
   * Лёгкая
   * Средняя
   * Сильная
 
 ### 13. Самообвинение / Чувство вины:
   * Нет
   * Незначительное
   * Среднее
   * Сильное
 
 ### 14. Психомоторное состояние:
   * Заторможенность
   * Нормальное
   * Возбуждённость
 
 ### 15. Когнитивный туман (Brain fog):
   * Чистая голова
   * Лёгкий туман
   * Средний
   * Сильный
 
 ### 16. Алкоголь / вещества:
   * Не употреблял
   * Употреблял немного
   * Употреблял много
    (Чтоб отследить связь между употреблением и спадом/улучшением)
